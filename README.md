# CI/CD Demo - Практична робота 7

Цей репозиторій є форком навчального репозиторію для відпрацювання мінімальної конфігурації Continuous Integration (CI) за допомогою GitHub Actions.

## Вимоги Базового Рівня (60 балів)

1.  Налаштовано CI-пайплайн у файлі `.github/workflows/ci.yml` з тригерами `push` та `pull_request`.
2.  Пайплайн включає кроки: `checkout`, `setup-node` (встановлення середовища), `npm install`, `npm test` (тести та лінт).
3.  Артефакт (звіт про покриття) завантажується за допомогою `upload-artifact`.
4.  Гарантовано успішне виконання щонайменше одного тесту.

---

## ⚙️ Локальний Запуск (Обов'язкові кроки)

Для локального запуску та тестування цього Node.js проєкту виконайте наступні кроки у терміналі:

1.  **Клонування репозиторію** (якщо ще не зроблено):
    ```bash
    git clone [https://github.com/Kiaru16/ci-cd-demo.git](https://github.com/Kiaru16/ci-cd-demo.git)
    cd ci-cd-demo
    ```
2.  **Встановлення залежностей**:
    ```bash
    npm install
    ```
3.  **Запуск тестів**:
    ```bash
    npm test
    ```
